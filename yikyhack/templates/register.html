Finding User ID

On Iphone
  1. Download iExplorer here: https://www.macroplant.com/iexplorer/
  2. Install and run
  3. Plug in iPhone
  4. Navigate to /User/Containers/Data/Application
  5. Search for the file: engineering.locus.chatter.plist
  6. Open file and scroll to the very bottom. Your userid will be shown
