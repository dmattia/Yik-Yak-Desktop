Finding User ID

On Iphone
  1. Download iExplorer here: https://www.macroplant.com/iexplorer/
  2. Install and run
  3. Plug in iPhone
  4. Navigate to /User/Containers/Data/Application
  5. Search for the file: engineering.locus.chatter.plist
    Note: If you are using a jailbroken phone and iFile, you must view as text, not through plist viewer
  6. Open file and scroll to the very bottom. Your userid will be shown
On Android
  1. Open file directory by plugging phone into computer and clicking view files
  2. Navigate to: /data/data/com.yik.yak/shared_prefs/com.yik.yak_preferences.xml
  3. Search for userid in the xml file
